Base class 기본 클래스

.btnBootstrap에는 패딩 및 콘텐츠 정렬과 같은 기본 스타일을 설정하는 기본 클래스가 있습니다 . 기본적으로 .btn컨트롤에는 투명한 테두리와 배경색이 있으며, 명시적인 포커스 및 호버 스타일이 없습니다.

이 .btn클래스는 버튼 변형과 함께 사용하거나 사용자 정의 스타일을 위한 기반으로 사용하도록 고안되었습니다.

.btn클래스를 단독으로 사용하는 경우 , 적어도 몇 가지 명시적 :focus및/또는 :focus-visible스타일을 정의하는 것을 잊지 마세요.

Variants 변형

부트스트랩에는 여러 가지 버튼 변형이 포함되어 있으며, 각각은 고유한 의미적 목적을 가지고 있으며, 더 많은 제어를 위해 몇 가지 추가 기능이 추가되었습니다.

접근성 팁: 색상을 사용하여 의미를 추가하면 시각적 표시만 제공되며, 화면 판독기와 같은 보조 기술 사용자에게는 전달되지 않습니다. 의미가 콘텐츠 자체에서 분명하게 드러나도록 하세요(예: 충분한 색상 대비가 있는 보이는 텍스트 ) 또는 클래스와 함께 숨겨진 추가 텍스트와 같은 대체 수단을 통해 포함되도록 하세요 .visually-hidden.

Disable text wrapping 텍스트 줄바꿈 비활성화

버튼 텍스트가 줄바꿈되지 않게 하려면, .text-nowrap버튼을 클래스에 추가할 수 있습니다. Sass에서는 $btn-white-space: nowrap각 버튼에 대해 텍스트 줄바꿈을 비활성화하도록 설정할 수 있습니다.

Button tags 버튼 태그

클래스 는 엘리먼트 .btn와 함께 사용하도록 설계되었습니다 . 그러나 이러한 클래스를 엘리먼트 button태그에 사용할 수도 있습니다 (일부 브라우저는 약간 다른 렌더링을 적용할 수 있음).a태그 input태그

a태그 현재 페이지 내의 새 페이지나 섹션으로 연결하는 대신, 페이지 내 기능(예: 콘텐츠 축소)을 트리거하는 데 사용되는 요소에 버튼 클래스를 사용하는 경우 이러한 링크에는 을 지정 role="button"하여 화면 판독기와 같은 보조 기술에 해당 목적을 적절히 전달해야 합니다.

Outline buttons 개요 버튼

버튼이 필요하지만, 그들이 제공하는 무거운 배경색은 필요 없나요? 기본 수정자 클래스를 .btn-outline-*모든 버튼의 배경 이미지와 색상을 제거하는 클래스로 바꾸세요.

일부 버튼 스타일은 비교적 밝은 전경색을 사용하므로 충분한 대비를 위해 어두운 배경에서만 사용해야 합니다.
